# Industry Newsletters & Blogs — Reading Instructions

This folder is intentionally empty at setup time. You populate it with content from product leaders that you want the OS to use as grounding truth for strategy and market context.

---

## Where the Content Lives

The primary newsletter and podcast archives are stored at:
- **Lenny's Newsletter + Podcast:** `../../lennys-newsletterpodcastdata-all/`
- **Substack content (Gergely Orosz, PM blogs):** `../../Substack content/`

The OS reads these directories directly when market context or PM frameworks are needed. You do not need to copy content here.

---

## When the OS Uses These Sources

Skills that automatically reference the newsletter archives:
- `/product-strategy [question]` — reads for frameworks and analogous case studies
- `/brainstorm [problem]` — reads for product patterns and mental models
- `/gtm-plan [feature]` — reads for GTM frameworks and launch playbooks
- `/presentation-draft [topic]` — reads for storytelling frameworks
- `/quick-start [topic]` — reads for market context before context library is populated

---

## How to Add Content Here

If you want to add specific articles or summaries to this folder for faster access:

1. Save as `.pdf` or `.md` (plain text preferred for search)
2. Name the file: `[author]-[topic]-[YYYY-MM].md` (e.g., `lenny-activation-metrics-2026-03.md`)
3. Add an entry to the index below
