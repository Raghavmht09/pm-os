## DSIM Competitive Landscape — Verified Intelligence (April 2026)
Source: Live web research (April 2026) + PM OS subagent research + DSIM_Competitive_BattleCards_1.pptx

### Threat Map
DSIM sits at the intersection of three markets — digital shelf analytics, retail media activation, and MMM/incrementality measurement. No single competitor owns all three.

| Tier | Competitors | Core tension with DSIM |
|---|---|---|
| Direct (HIGH) | CommerceIQ (Dec 2025: standalone measurement module), Incremental.com | Measurement-only positioning overlap; CIQ methodology weaker; Incremental.com uses shelf as noise, not output |
| DSA Incumbent (HIGH) | Profitero (Publicis; measurement via Pacvue partnership; Hofstetter departed March 2026) | Monitoring vs. causation; complement positioning |
| Retail Media (MEDIUM) | Pacvue (Amazon-first incrementality; Walmart gap), Skai (Walmart explicitly lags per G2) | Own activation; measurement is secondary and has shelf blind spot |
| MMM Pure-play (STRUCTURAL) | Analytic Partners, Circana, Measured, Haus | CFO conversation; slow; shelf-blind; DTC-first (Measured/Haus) |
| In-house (SPECIAL) | Walmart Connect | Free, first-party, deterministic — but media-only; no shelf decomposition |

### Critical 2025-2026 Market Events
1. **CommerceIQ standalone Incrementality Module** (Dec 2025): validates measurement-only market; but uses observational model (keyword/propensity); Walmart Connect publicly refuted this methodology type (April 2025)
2. **Walmart Connect Search Incrementality GA** (April 2025): randomized control testing on first-party POS data; Clorox beta customer; compresses third-party media measurement vendors but strengthens DSIM (shelf-native angle is untouched)
3. **Scintilla In-Store launch** (Feb 2026): Walmart Data Ventures store-level availability + void detection; strengthens DSIM's data input layer
4. **WPP Media names Incremental.com first "agency Connected Partner"** (Sept 2025): Incremental gets WPP scale; DSIM must track whether they add content quality or store-cluster features
5. **Profitero + Mars United Commerce = Profitero+** (May 2025): Sarah Hofstetter departed March 2026; Publicis fully in control; conflict-of-interest angle now leadership-structural
6. **Skai 2026 survey**: 75% of advertisers say incrementality is their #1 challenge; only 20% proficient — demand signal at scale
